▶ About HexScan

Built by hackers,
for everyone.

HexScan is a free collection of professional security tools. No login. No tracking. No paywalls. Just tools that work.

▶ The Project

HexScan started as a personal toolkit — a set of tools I kept building and rebuilding for my own security work. Password generators, header analyzers, breach checkers. Things I used daily but always had to jump between 5 different sites to access.


So I built them all in one place, made them free, and put them online. That's HexScan.
